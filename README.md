# Dubai Urban Heat Stress Analysis Overview

This repository provides a high-level overview of an ongoing research project focused on analyzing urban heat stress patterns in Dubai using satellite remote sensing and climate data.

The work is being carried out as part of a research internship under Dr. Rita L. Sousa at NYU Abu Dhabi. This repository is intended only as a portfolio-safe summary of the project and does not include code, datasets, or detailed internal workflows.

## Project Context

Urban heat stress is an important challenge in rapidly developing cities such as Dubai, where dense built-up areas, limited vegetation, and extreme climatic conditions can intensify thermal exposure.

This project explores how geospatial data can be used to better understand heat patterns across the city and support future analysis of urban thermal risk.

## Current Focus

The project currently involves building and testing workflows for generating key environmental layers over Dubai, including:

- vegetation-related indicators
- built-up area indicators
- land surface temperature
- near-surface air temperature
- relative humidity

These variables are being prepared to support later heat exposure analysis and spatial comparison across the city.

## High-Level Methodology

At a broad level, the project combines:

- satellite imagery for surface-based urban indicators
- climate reanalysis data for atmospheric variables
- geospatial processing for map generation and comparison
- exploratory heat exposure analysis to identify spatial patterns

The goal is to develop a structured workflow that can later support more advanced modeling, interpretation, and visualization.

## Project Goals

The broader goals of the project include:

- understanding how urban form relates to heat exposure
- identifying spatial patterns of thermal stress
- supporting future heat risk mapping and analysis
- contributing to data-driven urban and environmental research

## Tools Used

The project involves tools and platforms such as:

- Google Earth Engine
- USGS Earth Explorer
- Python
- satellite remote sensing data
- climate reanalysis data
- geospatial analysis workflows
